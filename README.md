# Hi, I'm Fabio Seidl

Data Engineer and Data Architect based in Florianópolis, Brazil. I've been in technology for 25+ years, the last 10+ working with data. These days I design data platforms and build distributed pipelines, mostly on Azure and Databricks, and I mentor engineers in data engineering, business intelligence, and data governance.

Outside of work you'll usually find me running LLMs locally, generating audiobooks with TTS models, 3D printing parts for my projects, or with an ESP32 wired into my off-road car's CAN bus. I also cave dive, which has nothing to do with any of this.

## Things I'm building

- [tts-audiobook-generator](https://github.com/fabioseidl/tts-audiobook-generator) — GPU-powered audiobook generator using XTTS v2, with voice cloning and a resumable pipeline
- [mitsubish-pajero-dashboard](https://github.com/fabioseidl/mitsubish-pajero-dashboard) — distributed ESP32 dashboard for my Pajero Dakar 3.2. A server node reads OBD-II/CAN data and wireless nodes render it on multiple displays

- [mpbr-price-alert](https://github.com/fabioseidl/mpbr-price-alert) — price drop and target price alerts using the Menor Preço Brasil service
- [baby-health-monitor](https://github.com/fabioseidl/baby-health-monitor) — ESP32 firmware developed by a nerd father to monitor when his daughter sleeps
